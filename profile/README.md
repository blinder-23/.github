# 시각장애인 학생들을 위한 학사정보 앱

읽기도 찾기도 힘들었던 점자 식단표, 이제 블린더로 읽어보세요! 언제나 궁금한 점심 메뉴, 우리 학교 학사일정을 블린더에서 모두 찾아볼 수 있습니다.
